# 2. 서비스 개요

---

> **서비스명**
> 

데파Defa

> **한줄 소개**
> 

업무효율성 향상을 위한 데이터셋 서치 AI 에이전트 + 초안작성기

> **개발 동기**
> 

디지털 시대의 모든 기획은 데이터에서 시작됩니다. 

하지만 현실에서 데이터를 찾는 과정은 아이디어 도출보다 더 많은 시간을 소모하게 됩니다. 어떤 데이터를 써야 하는지, 어디에 있는지, 어떻게 활용할 수 있는지조차 알기 어려워 많은 프로젝트가 초반 단계에서 멈춰버리곤 합니다.

저희는 해커톤에서 팀 주제를 정하거나 개인 프로젝트를 할 때도 데이터를 찾는데에서 많이 멈칫거리곤 했습니다. 그 경험을 살려 다른사람들도 연구 과제를 준비할 때, 공공·산업 데이터를 수집하는 일이 막막하고 반복적이라고 생각하지 않을까요? 우리는 문제를 해결하고 싶은데, 시작조차 못한다는 좌절을 반복했습니다.

산업·청년·관광·환경 등 수많은 데이터는 여기저기 흩여져있고 현장 실무자는 물론 학생·창업자·연구자 모두 “데이터를 찾는 법”을 모른다는 동일한 장애물을 가지고 있습니다.

그래서 우리는 “**아이디어나 문제를 정의하면, 필요한 데이터를 대신 찾아주고 그 데이터를 기반으로 실행가능한 초안을 만들어주는 도구를 만들자**”고 생각했습니다.

데이터파인더, Defa는 단순한 검색기가 아니라 데이터 탐색을 시작점부터 자동화하는 AI 파트너입니다. “부산 청년 문제 해결, “울산 제조업 인력난”, “경남 관광 데이터” 처럼 문제만 제시하면 됩니다. 

우리는 이 서비스를 통해 데이터 접근의 장벽을 낮추고, 누구나 문제 해결자가 될 수 있는 환경을 만들고자 합니다. 아이디어가 사라지는 이유가 정보 부족이 아니라 **행동으로 이어지지 못하는 시스템 때문이라면**, Defa는 그 첫 단계를 대신 만들어주며 여러분과 함께 하겠습니다.

# 4. 기대 효과

서비스를 **통해 얻을 수 이점, 발전 방향성 등을 알려주세요.**

---

## 🏛️ 1. **사회적 기대효과**

### ① 데이터 접근 장벽 해소

- 공공데이터·통계·기관 자료 등 실무자·시민·학생이 접근하기 어려운 자료를 **AI가 직접 찾아 정리**.
- **정보격차** → **문제해결 능력 격차**를 줄여 **기회 평준화**를 실현.

### ② 지역 문제 해결 역량 강화

- 부울경(부산·울산·경남) 특성에 맞는 사회문제(청년, 산업, 관광, 환경)에 대해
    
    **누구나 데이터 기반으로 문제정의 → 솔루션 설계 → 제안서 제작** 가능.
    
- *지역 주민도 “문제 제기자”가 아니라 “문제해결자”가 될 수 있음.

### ③ 공공참여 확대

- 공공기관 정책·사업 평가 시, **데이터 기반 관점**을 시민이 쉽게 사용할 수 있음.
- 행정 데이터 활용이 전문가/연구소에 집중되는 현상을 완화.

> 👉 “데이터 민주화(Data Democratization)”를 통한 사회적 혁신 촉진
> 

---

## 💰 2. **경제적 기대효과**

### ① 업무 효율성 증대 → 비용 절감

- 기획/보고/연구 초안 작성에 들어가는 **준비 시간 50~80% 절감**.
- 반복적 자료 수집·정리 업무를 대체하여 **실제 생산 활동에 집중**.

### ② 스타트업·중소사업자의 시장 진입 장벽 감소

- 초기 시장조사, 경쟁 분석, 상권조사, 정책근거 수집 시간을 단축.
- **아이디어 → 프로토타입 → 기획서 → 투자 대응** 흐름 가속화.

### ③ 데이터 기반 의사결정 → 사업 실패 리스크 감소

- “감”이나 “경험”이 아닌 **실제 데이터 기반 판단**.
- 예산 배분/마케팅/정책 기획이 더 과학적으로 개선.

> 👉 동일 인력·동일 시간 대비 더 높은 경제적 아웃풋을 생성하는 구조
> 

---

## 🧠 3. **지식·교육적 기대효과**

### ① 데이터 리터러시 향상

- 학생·예비 스타트업·공공 실무자가
    
    “데이터 → 해석 → 문서화” 과정을 자연스럽게 경험.
    

### ② 연구/학습 기반 확장

- 논문 서론/리뷰/조사 보고서 등을 **AI 기반으로 자동 초안 생성**.
- 데이터에 접근하는 방식이 익숙해져 **다음 단계 연구의 난이도 감소**.

### ③ 실전형 학습 문화 조성

- “아이디어 기반 학습”이 아니라 “근거 기반 학습”으로 전환.
- 캡스톤/해커톤/창업동아리 등 실무 프로젝트 참여자 역량 상승.

---

## 🤖 4. **기술적 기대효과**

### ① 공공·산업 데이터의 활용 가치 증대

- 흩어져 있던 데이터를 **AI가 탐색 → 사용자가 이해 가능한 형태 → 목적별 문서로 재구성**.
- 기존 데이터는 “저장된 정보” → Defa는 “활용 가능한 자산”으로 변환.

### ② 실행 자동화 생태계 확장

- 이후 GPT/LLM API 적용 시:
    - 데이터 분석
    - 리포트 생성
    - 시각화
    - 정책 문서 자동 보완
- 단순 “검색챗봇 → 문서 자동화 엔진”으로 고도화 가능.

### ③ API·RAG·브라우저 OCR 확장

- 향후 **문서 업로드 → 데이터 추출 → PDF 결과물 생성**까지 확장 가능.
- 교육·행정·연구 등 범용 플랫폼으로 수평 확장.
